## Phase 4: Chat Performance (Commits 16-20)

**Problem**: Chat takes ~10s to respond
**Impact**: Poor user experience

### Commits

16. `perf(chat): add request timeout configuration`
    - Configurable timeout (default 30s)
    - Show progress after 3s

17. `perf(chat): implement response caching`
    - Cache common queries (5min TTL)
    - Cache key based on agent + message hash

18. `perf(chat): add optimistic UI improvements`
    - Show typing indicator immediately
    - Progressive message rendering

19. `feat(chat): add slow response indicator`
    - "Taking longer than usual" message after 5s
    - Estimated time remaining

20. `perf(chat): implement request debouncing`
    - Prevent duplicate requests
    - Cancel previous pending requests
