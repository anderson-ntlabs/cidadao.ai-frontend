# Cidadão.AI Academy

> **Feito por alunos, para alunos**

Plataforma gamificada de estágio em parceria com o IFSULDEMINAS/LabSoft e Neural Thinker AI Engineering.

## Sobre o Programa

O Cidadão.AI Academy é um programa de estágio inovador que combina:

- **Projeto Real**: Contribua para o Cidadão.AI, uma plataforma de transparência governamental em produção
- **Gamificação**: Ganhe XP, badges e suba no ranking enquanto aprende
- **Flexibilidade**: Participe de acordo com sua disponibilidade
- **Mentoria**: Receba orientação de profissionais experientes
- **Prêmios**: O 1º lugar ganha vaga de estágio na Neural Thinker

## Trilhas Disponíveis

| Trilha       | Tecnologias                     | Nível            |
| ------------ | ------------------------------- | ---------------- |
| **Backend**  | Python, FastAPI, PostgreSQL     | Básico em Python |
| **Frontend** | Next.js, React, Tailwind        | Básico em JS/CSS |
| **IA/ML**    | DSPy, LangChain, Transformers   | Intermediário    |
| **DevOps**   | Docker, GitHub Actions, Grafana | Básico em Linux  |

## Sistema de Progressão

```
Novato (0-100 XP) → Aprendiz (100-500 XP) → Contribuidor (500-2000 XP) → Mentor (2000-5000 XP) → Arquiteto (5000+ XP)
```

## Como Participar

1. Fork do repositório [cidadao.ai-backend](https://github.com/anderson-ufrj/cidadao.ai-backend)
2. Cadastre-se na plataforma Academy
3. Escolha sua trilha
4. Complete missões e ganhe XP
5. Suba no ranking!
