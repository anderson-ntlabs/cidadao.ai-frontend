## 2. Missão

### 2.1 Declaração de Missão

> **"Democratizar o acesso à educação tecnológica de qualidade através de uma experiência de aprendizagem prática, gamificada e culturalmente relevante, capacitando brasileiros para contribuir com projetos de impacto social."**

### 2.2 Pilares da Missão

```
                    MISSÃO ÁGORA
                        |
     +------------------+------------------+
     |                  |                  |
DEMOCRATIZAR      CAPACITAR          IMPACTAR
     |                  |                  |
- Gratuito         - Prática real     - Open source
- Acessível        - Gamificação      - Transparência
- Inclusivo        - Mentoria IA      - Cidadania
```

#### Pilar 1: Democratizar

| Aspecto    | Implementação                    |
| ---------- | -------------------------------- |
| Financeiro | Plataforma 100% gratuita         |
| Geográfico | Acesso remoto, mobile-first      |
| Social     | Sem pré-requisitos formais       |
| Físico     | Acessibilidade WCAG AAA, VLibras |

#### Pilar 2: Capacitar

| Aspecto        | Implementação                            |
| -------------- | ---------------------------------------- |
| Técnico        | Trilhas em Backend, Frontend, IA, DevOps |
| Prático        | Contribuição real em projeto de produção |
| Comportamental | Soft skills via colaboração open-source  |
| Contínuo       | Sistema de progressão e especialização   |

#### Pilar 3: Impactar

| Aspecto      | Implementação                             |
| ------------ | ----------------------------------------- |
| Individual   | Portfólio de contribuições verificáveis   |
| Social       | Plataforma de transparência governamental |
| Profissional | Conexão com oportunidades de emprego      |
| Cívico       | Fortalecimento da cidadania digital       |

## 7. Diferenciais Competitivos

### 7.1 Versus Cursos Online Tradicionais

| Aspecto    | Cursos Tradicionais  | Ágora Academy        |
| ---------- | -------------------- | -------------------- |
| Preço      | R$ 500-5000          | Gratuito             |
| Prática    | Exercícios simulados | Projeto real         |
| Portfólio  | Certificado genérico | PRs no GitHub        |
| Networking | Fóruns isolados      | Ranking + comunidade |
| Relevância | Conteúdo pré-gravado | Projeto em evolução  |

### 7.2 Versus Bootcamps

| Aspecto  | Bootcamps            | Ágora Academy     |
| -------- | -------------------- | ----------------- |
| Duração  | 3-6 meses intensivos | Flexível          |
| Custo    | R$ 10.000-30.000     | Gratuito          |
| Horário  | Full-time            | Assíncrono        |
| Suporte  | Professores humanos  | Mentores IA 24/7  |
| Garantia | Emprego ou devolução | Prêmio de estágio |

### 7.3 Versus Contribuição Open Source Direta

| Aspecto        | OSS Direto            | Ágora Academy           |
| -------------- | --------------------- | ----------------------- |
| Onboarding     | Confuso, intimidante  | Guiado, gamificado      |
| Suporte        | Depende da comunidade | Mentores IA sempre      |
| Progressão     | Não estruturada       | Trilhas claras          |
| Reconhecimento | Apenas commits        | XP, badges, certificado |

## 8. Compromissos

### 8.1 Compromisso com Estudantes

> "Todo estudante da Ágora terá acesso igual a oportunidades de aprendizagem e reconhecimento, independentemente de sua origem, recursos financeiros ou localização geográfica."

### 8.2 Compromisso com Qualidade

> "Manteremos os mais altos padrões de qualidade em conteúdo, código e experiência do usuário, buscando continuamente feedback e melhorias."

### 8.3 Compromisso com Acessibilidade

> "A plataforma será acessível a todos, seguindo WCAG AAA e oferecendo suporte a LIBRAS através do VLibras."

### 8.4 Compromisso com Transparência

> "Assim como o projeto Cidadão.AI promove transparência governamental, a Ágora será transparente em seus processos, métricas e decisões."
