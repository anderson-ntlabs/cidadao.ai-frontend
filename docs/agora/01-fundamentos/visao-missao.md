## 2. Missao

### 2.1 Declaracao de Missao

> **"Democratizar o acesso a educacao tecnologica de qualidade atraves de uma experiencia de aprendizagem pratica, gamificada e culturalmente relevante, capacitando brasileiros para contribuir com projetos de impacto social."**

### 2.2 Pilares da Missao

```
                    MISSAO AGORA
                        |
     +------------------+------------------+
     |                  |                  |
DEMOCRATIZAR      CAPACITAR          IMPACTAR
     |                  |                  |
- Gratuito         - Pratica real     - Open source
- Acessivel        - Gamificacao      - Transparencia
- Inclusivo        - Mentoria IA      - Cidadania
```

#### Pilar 1: Democratizar

| Aspecto    | Implementacao                    |
| ---------- | -------------------------------- |
| Financeiro | Plataforma 100% gratuita         |
| Geografico | Acesso remoto, mobile-first      |
| Social     | Sem pre-requisitos formais       |
| Fisico     | Acessibilidade WCAG AAA, VLibras |

#### Pilar 2: Capacitar

| Aspecto        | Implementacao                            |
| -------------- | ---------------------------------------- |
| Tecnico        | Trilhas em Backend, Frontend, IA, DevOps |
| Pratico        | Contribuicao real em projeto de producao |
| Comportamental | Soft skills via colaboracao open-source  |
| Continuo       | Sistema de progressao e especializacao   |

#### Pilar 3: Impactar

| Aspecto      | Implementacao                             |
| ------------ | ----------------------------------------- |
| Individual   | Portfolio de contribuicoes verificaveis   |
| Social       | Plataforma de transparencia governamental |
| Profissional | Conexao com oportunidades de emprego      |
| Civico       | Fortalecimento da cidadania digital       |

## 7. Diferenciais Competitivos

### 7.1 Versus Cursos Online Tradicionais

| Aspecto    | Cursos Tradicionais  | Agora Academy        |
| ---------- | -------------------- | -------------------- |
| Preco      | R$ 500-5000          | Gratuito             |
| Pratica    | Exercicios simulados | Projeto real         |
| Portfolio  | Certificado generico | PRs no GitHub        |
| Networking | Foruns isolados      | Ranking + comunidade |
| Relevancia | Conteudo pre-gravado | Projeto em evolucao  |

### 7.2 Versus Bootcamps

| Aspecto  | Bootcamps            | Agora Academy     |
| -------- | -------------------- | ----------------- |
| Duracao  | 3-6 meses intensivos | Flexivel          |
| Custo    | R$ 10.000-30.000     | Gratuito          |
| Horario  | Full-time            | Assincrono        |
| Suporte  | Professores humanos  | Mentores IA 24/7  |
| Garantia | Emprego ou devolucao | Premio de estagio |

### 7.3 Versus Contribuicao Open Source Direta

| Aspecto        | OSS Direto            | Agora Academy           |
| -------------- | --------------------- | ----------------------- |
| Onboarding     | Confuso, intimidante  | Guiado, gamificado      |
| Suporte        | Depende da comunidade | Mentores IA sempre      |
| Progressao     | Nao estruturada       | Trilhas claras          |
| Reconhecimento | Apenas commits        | XP, badges, certificado |

## 8. Compromissos

### 8.1 Compromisso com Estudantes

> "Todo estudante da Agora tera acesso igual a oportunidades de aprendizagem e reconhecimento, independentemente de sua origem, recursos financeiros ou localizacao geografica."

### 8.2 Compromisso com Qualidade

> "Manteremos os mais altos padroes de qualidade em conteudo, codigo e experiencia do usuario, buscando continuamente feedback e melhorias."

### 8.3 Compromisso com Acessibilidade

> "A plataforma sera acessivel a todos, seguindo WCAG AAA e oferecendo suporte a LIBRAS atraves do VLibras."

### 8.4 Compromisso com Transparencia

> "Assim como o projeto Cidadao.AI promove transparencia governamental, a Agora sera transparente em seus processos, metricas e decisoes."
